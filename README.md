1. TECH STACK

Basic Interview Simulator using AI to track the score.
Frontend:-
UI is designed using React.js.
JavaScript is used with react.js to connect backend API.
HTML & CSS for structure and enhacement of the webpage.

Backend:-
Python is used as primary programming language for backend
FastAPI (REST API) for framework

Basic NLP / scoring logic (custom or rule-based)


2. FEATURES

Real-time interview question system
Answer submission with API integration
AI-based scoring system
Feedback generation for each answer
Overall performance score calculation
Clean UI with dynamic updates
Input validation (prevents empty submission)
Login system for Student and Admin seperately with Logout option to went back to Login Page
Admin Login Panel which alter several features for admin like clearing leaderboard,clearing history,Adding questions to any Topics


3. ARCHITECTURE

Frontend (React) communicates with backend via REST API
Backend (FastAPI) processes answers and returns:
Score
Feedback
State management handled using React Hooks
